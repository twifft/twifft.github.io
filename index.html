Privacy Policy

Last updated: 2024-01-01

This Privacy Policy explains how DailyClipzTikTok (“we”, “our”, “the App”) handles data when using our application.

1. Data We Collect

DailyClipzTikTok does not collect personal data from users outside of what is provided by the TikTok Developer API. TikTok may provide:

  • Basic profile information
  • Permissions granted by the user
  • Video upload access tokens

2. How We Use Your Data

We only use TikTok-provided data for:

  • Uploading videos on the user’s behalf
  • Managing captions and upload parameters

3. AI-Generated Content

All videos are generated using automated tools and AI. No personal user data is used in the creation process.

4. Data Storage

We do not store any personal TikTok data on external servers. Temporary tokens are only used during the upload process.

5. Sharing of Data

We do not share data with any third parties.

6. User Rights

Users may revoke API permissions at any time through TikTok's settings.

7. Security

We use standard technical measures to ensure data is handled securely.
